## Reporting a Vulnerability

We take the security of **Cyber Brother** seriously. If you have found a vulnerability, please help us fix it by following these steps:

1.  **Do NOT open a public issue.** This allows hackers to exploit the bug before we fix it.
2.  Please send a detailed report via email to: **shohjahon@example.com** (or your private contact).
3.  Include steps to reproduce the issue.

### Our Promise
*   We will review your report within 24 hours.
*   We will acknowledge your contribution.
*   We will release a patch as soon as possible.
